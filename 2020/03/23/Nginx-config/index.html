Load Balancing 的幾種方式

  • Round Robin(default): 可以在IP/domain設定weight權重，依照權重分配
  • Least Connections: 流量會導到連結最少的那個server
  • IP Hash: 會將IPv4的第一組八位元或是IPv6整串經過hash table對應到一台server，只要使用者從同一個IP來，流量就會固定對應到同一台server。
